# 🌤️ WeatherNow

> **Real-Time Weather Forecast at Your Fingertips**

WeatherNow is a modern and responsive weather application built with **React.js** and **Vite**. It provides real-time weather updates, detailed forecasts, and location-based weather information through a clean, fast, and user-friendly interface.

---

## 🚀 Features

- 🔍 Search weather by city
- 📍 Current location weather
- 🌡️ Real-time temperature
- 🌤️ Weather conditions with dynamic icons
- 💧 Humidity
- 🌬️ Wind speed
- 👀 Visibility
- 🌅 Sunrise & Sunset
- 📅 Hourly forecast
- 📆 7-day weather forecast
- 🌫️ Air Quality Index (AQI)
- ⭐ Favorite cities
- 🕒 Search history
- 🌙 Dark / Light mode
- 🌍 Celsius / Fahrenheit switch
- 📱 Fully responsive design
- ⚡ Loading animations
- ❌ Error handling
- 🎨 Smooth animations using Framer Motion

---

## 🛠️ Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS v4
- Axios
- Framer Motion
- React Icons
- React Hot Toast

### APIs

- Open-Meteo API *(or OpenWeather API, depending on what you use)*

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/WeatherNow.git
```

Move into the project

```bash
cd WeatherNow
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

---

## 🌐 Environment Variables

Create a `.env` file in the project root.

```env
VITE_WEATHER_API_KEY=YOUR_API_KEY
```
